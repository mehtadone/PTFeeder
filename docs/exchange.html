# COIN-M Futures

For coin futures on binance with feeder you need to set:

  • Basecurrency to BTC //or DOT etc depending on what base currency you want to accumulate
  • BaseCoinPair to BTCUSD_PERP
  • Enabled coins should be PERP if you only intend to trend the PERP coin and not quarterly
